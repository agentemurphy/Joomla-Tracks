Joomla-Tracks by julienV

Racing league extension for Joomla

Joomla Tracks is a racing league management extension for Joomla 1.5. In fact, it works for every kind of competition were you earn points for each round according to your position.

Dependencies

needs Joomla 1.5

Install

use joomla backend install from folder, or create archives accordingly beforehand.
